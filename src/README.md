# Mergington High School Activities

A website application that allows teachers to manage student sign-ups for extracurricular activities at Mergington High School.

## Features

- View all available extracurricular activities
- Filter activities by category (Sports, Arts, Academic, Community, Technology)
- Filter activities by day of the week
- Filter activities by time range (morning, afternoon, or weekend)
- Search activities by name, description, or schedule
- View enrolled participants and available spots with a capacity indicator
- Teacher login and logout
- Register students for activities (requires teacher login)
- Unregister students from activities (requires teacher login)
